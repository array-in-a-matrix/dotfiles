# Dotfiles
I use 2 DE/WM on my Arch Linux system:
   - KDE/kwin
   - SwayWM/wlroots

I also use `lightDM` as my display manager.

Dotfiles managed using [chezmoi](https://www.chezmoi.io/).

## SwayWM
- Terminal: `Alacritty`
- File Manager: `LF`
- Bar: `Waybar`
- Launcher: `Rofi`

## KDE
- Terminal: `Konsole`
- File Manager: `Dolphin`
